# Web frameworks - Flask, FastAPI

## Læringsmål

* Kunne oprette et API skrevet i `Flask`.    
* Kunne oprette et API skrevet i `FastAPI`.    
* Forstå og bruge pythons type hints
* Forstå forskellen på `sync` og `async` funktioner
* Kunne oprette et og køre projekt med `uv`

## Forberedelse

Skim følgende sider igennem

* [Flask Quickstart](https://flask.palletsprojects.com/en/stable/quickstart/#a-minimal-application)
* [Fast API - Learn](https://fastapi.tiangolo.com/learn/)

## Dagen i dag

Undervisning i dag handler overordnet om at kunne arbejde med backend JSON API'er.      
Vi kigger på 2 forskellige frameworks: først `Flask` og herefter `FastAPI`, og den database vi kommer til at bruge er en `sqlite3` database.
